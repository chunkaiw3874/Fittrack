## 1. App 概述

FitTrack 是一個專門為居家健身設計的 PWA (網頁應用)，部署在 GitHub Pages 上。

**特色：**

- ✅ 完全離線可用（資料存在 iPhone 本地）
- ✅ 跟原生 App 一樣的體驗（加入主畫面）
- ✅ 不收集任何個人資料
- ✅ 永久免費
- ✅ 103 個內建動作資料庫
- ✅ 中英文動作名稱、YouTube 教學連結

**部署位置：**

- 程式碼：https://github.com/chunkaiw3874/Fittrack
- 線上 App：https://chunkaiw3874.github.io/Fittrack/

## 2. 快速上手

### 第一次安裝

1. 用 **Safari** 開啟：`https://chunkaiw3874.github.io/Fittrack/`
1. 點底部「分享」按鈕（⬆️）
1. 選「加入主畫面」
1. 主畫面會出現「FitTrack」圖示
1. 點開即可使用

### 更新版本

每次有新版本部署到 GitHub 時：

1. 主畫面長按 FitTrack → 移除 App
1. 設定 → Safari → 清除瀏覽記錄與網站資料
1. Safari 重新打開 `https://chunkaiw3874.github.io/Fittrack/?v=新版本號`
1. 重新加入主畫面

## 3. 主要功能

### 3.1 首頁

打開 App 看到的第一個畫面：

```
┌─────────────────────────────┐
│ Kevin 的健身計畫              │
│                             │
│ 當前設定                     │
│ 💪 標準                     │  ← 點選改強度
│ 30-40分鐘 · 4動作 × 3組      │
│                             │
│ 推薦 - 今日訓練               │
│ [星期一: 上肢推 + 代謝]      │  ← 點開始訓練
│                             │
│ 所有訓練日                    │
│ [上肢推] [下肢] [上肢拉] [核心]│
│                             │
│ 📚 動作資料庫                │  ← 搜尋 103 個動作
│ 📊 查看進度                  │  ← 看訓練紀錄
│                             │
├─────────────────────────────┤
│  🏋️ 健身    🥗 健康餐       │  ← 底部分頁
└─────────────────────────────┘
```

### 3.2 訓練強度設定

3 種強度可選（點首頁的「當前設定」進入）：

|強度  |動作數|組數|時間      |適合     |
|----|---|--|--------|-------|
|⚡ 快速|3  |2 |15-20 分鐘|時間緊、輕度日|
|💪 標準|4  |3 |30-40 分鐘|一般訓練日  |
|🔥 深度|5  |4 |45-60 分鐘|週末加練   |

### 3.3 訓練模板

App 內建 4 個訓練模板：

|訓練日|名稱      |重點     |動作                    |
|---|--------|-------|----------------------|
|一  |上肢推 + 代謝|胸、肩、三頭 |啞鈴胸推、肩推、伏地挺身、側平舉、三頭伸展 |
|二  |下肢 + 代謝 |腿、臀    |啞鈴深蹲、保加利亞分腿蹲、硬舉、臀橋、弓步蹲|
|四  |上肢拉 + 全身|背、二頭、全身|啞鈴划船、單臂划船、二頭彎舉、抓舉、錘式彎舉|
|五  |核心 + 活動度|核心、恢復  |平板支撐、死蟲式、鳥狗式、側平板、捲腹   |


> 三/六/日是休息日

### 3.4 自訂訓練

1. 點任一訓練日的「自訂」按鈕
1. 點動作右邊的「替換」可換成同類別的其他動作
1. 自動儲存

### 3.5 開始訓練

#### 次數動作流程（如：啞鈴胸推 10-12 下）

```
1. 點訓練日的「開始訓練」按鈕
2. 看到當前動作的詳細資料
3. 完成第 1 組動作（你自己做）
4. 點「第 1 組」按鈕
5. 跳出彈窗：「第 1 組完成幾下？」
6. 輸入次數（例：12）
7. 點「記錄」
8. 第 1 組變綠 ✓，第 2 組高亮
9. （可選）點「⏱ 想要休息計時？」開始休息計時
10. 完成第 2、3 組...
11. 全部完成 → 「下一個動作」按鈕
```

#### 秒數動作流程（如：平板支撐 30 秒）

```
1. 進入動作頁面
2. 看到綠色大按鈕「開始 30 秒」
3. 按下！（一次！）
4. 紅色畫面：動作中 30...29...28
5. 30 秒倒數結束 → 自動進入休息
6. 藍色畫面：休息中 30...29
7. 倒數結束 → 自動進入準備
8. 黃色畫面：3...2...1
9. 自動開始第 2 組
10. 全部完成 → 動作完成
```

詳見 [自動計時模式](#5-自動計時模式)。

## 4. 動作資料庫

### 4.1 統計

- **總動作數：103 個**
- **覆蓋肌群**：胸、背、肩、腿、臀、手臂、核心、全身、心肺
- **可用器材**：啞鈴、徒手、阻力帶、瑜伽墊、跳繩

### 4.2 進入動作資料庫

從首頁點「📚 動作資料庫」按鈕。

### 4.3 搜尋與篩選

```
┌─────────────────────────────┐
│ 🔎 [搜尋動作名稱...]         │
│                             │
│ 💪 訓練肌群（多選）          │
│ [胸][背][肩][腿][臀]         │
│ [手臂][核心][全身][心肺]     │
│                             │
│ 🛠️ 需要器材（多選）          │
│ [啞鈴][徒手][阻力帶]         │
│ [瑜伽墊][跳繩]               │
│                             │
│ ⭐ 難度（多選）               │
│ [★初階] [★★中階] [★★★進階] │
│                             │
│ 📋 找到 X 個動作              │
│ ...動作列表...                │
└─────────────────────────────┘
```

**篩選邏輯：**

- 同類別內：選中任一即顯示（OR）
- 不同類別之間：必須同時符合（AND）
- 例：選「胸」+「啞鈴」= 用啞鈴練胸的動作

### 4.4 收藏動作

點動作旁的 ☆ → 變成 ★ 加入收藏。

收藏會自動儲存到 iPhone，下次打開還在。

### 4.5 查看動作詳情

點動作的「👁 查看詳情 + 影片」彈出詳情頁：

- 訓練肌肉（主要 + 次要）
- 需要器材
- 次數範圍 + 休息秒數
- 📺 觀看示範影片（YouTube）
- ✅ 正確做法（步驟）
- ⚠️ 注意事項

## 5. 自動計時模式

**只適用於秒數動作**（平板支撐、開合跳、登山者等）

### 5.1 流程圖

```
按下「開始 30 秒」
        ↓
🟥 動作中 (30秒倒數)
        ↓ (倒數結束 → 自動)
🟦 休息中 (30秒倒數)
        ↓ (倒數結束 → 自動)
🟨 準備中 (3秒倒數)
        ↓ (倒數結束 → 自動)
🟥 動作中 (第2組 30秒)
        ↓ (重複...)
✅ 完成
```

### 5.2 控制按鈕

每個階段都有 2 個按鈕：

- **跳過**：直接進入下一階段
- **停止**：完全停止自動模式（回到「開始」按鈕）

### 5.3 提醒設定

設定頁面可開關：

- 🔔 **聲音提醒**（預設開）：倒數最後 3 秒「嗶嗶嗶」
- 📳 **震動提醒**（預設關）：iPhone 震動

## 7. 常見問題

### Q1：載入卡很久怎麼辦？

A：通常 3-5 秒內會載入完。如果超過 10 秒：

1. 重新整理頁面
1. 清除 Safari 快取
1. 移除主畫面 App 重新加入

### Q2：資料會不會不見？

A：資料儲存在 iPhone 的 localStorage：

- ✅ App 關了還在
- ✅ 手機重開機還在
- ⚠️ 清除 Safari 網站資料 → 會清空
- ⚠️ 移除 App 圖示 → 不會清，重新加回去資料還在

**建議：每月截圖「進度」頁面備份。**

### Q3：訓練紀錄沒有出現？

A：檢查項目：

1. 訓練時要記得點「第 X 組」記錄次數（不是只做完動作）
1. 進度頁面只顯示有完成記錄的訓練
1. 如果還是沒紀錄，告訴我，可能是 bug

### Q4：無法從第一組進入第二組？

A：訓練時：

1. 完成第 1 組動作（你自己做）
1. **點藍色「第 1 組」按鈕**
1. 跳出彈窗 → 輸入次數 → 「記錄」
1. 第 1 組變綠 ✓，第 2 組會高亮閃爍

如果按鈕沒反應，告訴我具體狀況。

### Q5：影片不對怎麼辦？

A：影片是用 YouTube 搜尋連結，每次跳出來的影片可能不同。
如果某動作的搜尋結果不準，告訴我動作名稱，我會調整搜尋詞。

### Q6：自動模式按鈕沒反應？

A：檢查：

1. 那個動作是「秒數動作」嗎？（如平板支撐）
1. 次數動作（如啞鈴胸推）不會有自動模式
1. 如果是秒數動作但按了沒反應，告訴我

### Q7：可以取消休息計時嗎？

A：v10.0 開始：

- 次數動作的休息計時改為「可選」（預設關閉）
- 想要計時可點「⏱ 想要休息計時？」按鈕
- 不想計時就直接做下一組

### Q8：聲音不響？

A：iPhone 的限制：

- ✅ App 開啟狀態 → 聲音正常
- ❌ App 在背景 → 聲音可能停止
- ✅ 設定靜音模式 → 聲音不會響（這是正常的）

訓練時建議讓 App 保持開啟。
